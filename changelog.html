Commands Documentation

v2.0

Features:

  • Added tp to vehicle command: ?tp2v
  • Customizable roles that give specific users specific access
  • Added a new command to clear radiation: ?clearRadiation.
  • You can now call commands using buttons! Name your button like so: ?cc tpl beginner (this will teleport the player pressing the button to beginner island if they have permission to).
  • Server admins can now set a max_mass preference that will remove all vehicles from the world that are too heavy, to help keep servers stable and performant.

Changes:

  • Removed underscores from command names, in preference of "camel-casing" them so they can be typed easier.
  • Made parent tables to hold functions to make code neater (example: player.kick()).
  • More commands will assume self is target when no target player is specified. Also added "me" argumement option to many commands.
  • As many commands as possible can now accept a playerID instead of a peer_id
  • Split the old tp command into many new tp commands
  • Renamed ?tp_vehicle to ?tpv
  • server.save() is called after making changes to persistent data. Unfortunately this only works for dedicated servers.

Fixes:

  • Fixed the give command giving used/empty equipment after v1.2.5-6 update for Stormworks.
  • The equip_on_respawn preference would replace some equipment when rejoining the server.
  • Adjusted the telport commands so players are less likely to fall through the terrain due to the game's collisions taking too long to load.
  • All help info was moved into the ?ccHelp command in order to increase compatibility with other addons.

v1.02

Fixes:

  • Fixed issue where dedicated server owners could not be made owners.

v1.01

Fixes:

  • Fixed error when attempting to heal a player that is not on the server.
  • Corrected punctuation for ?player_perms command.
  • Disabling "map teleport" ("Allow Teleport" in the custom options menu) prevents non admin/owner players from teleporting.
  • Added ?deny_tp command that blocks another player from teleporting to you (Admins can override this).
