Preferences

Name Default Value Type Description
equipOnRespawn Boolean If players should be equipped with the start equipment when they spawn.
keepInventory Boolean If players' inventory should be preserved when they die.
removeVehicleOnLeave Boolean If a player's vehicles should be removed when they leave the server. The vehicles will only be removed when unloaded.
maxVoxels 0 Number The maximum number of voxels a vehicle can take up. Vehicles larger than this will not be able to be spawned.
startEquipmentA 0 Number The ID of the equipment to place in slot A for players.
startEquipmentB 15 Number The ID of the equipment to place in slot B for players.
startEquipmentC 6 Number The ID of the equipment to place in slot C for players.
startEquipmentD 11 Number The ID of the equipment to place in slot D for players.
startEquipmentE 0 Number The ID of the equipment to place in slot E for players.
startEquipmentF 0 Number The ID of the equipment to place in slot F for players.
welcomeNew Text The welcome message that should be shown to new players.
welcomeReturning Text The welcome message that should be shown to returning players.
companion Boolean If Carsa's Companion is active and should be used.
companionInformationOnJoin Boolean If players should be informed how they can connect to Carsa's Companion.
